# Connect Four AI Game Project

## Overview
Connect Four is an academic Algorithms project implemented using C++.

The project is a Connect Four game with an AI opponent that analyzes moves and chooses the best action using different algorithms.

## Features
- Player vs AI mode
- AI vs AI mode
- 6x7 game board
- Input validation
- Win and draw detection
- Intelligent AI moves

## Algorithms Used
- Minimax algorithm
- Alpha-Beta pruning
- Greedy heuristic evaluation
- Dynamic Programming using Transposition Table (state caching)

## Project Goal
The goal of this project is to apply algorithms concepts in C++ to build an intelligent game-playing system.
